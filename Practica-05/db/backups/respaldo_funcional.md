# Respaldo Funcional

Esta sección describe el propósito y contenido de cada archivo de respaldo incluido en el proyecto.

---

###  Descripción
Contiene los datos funcionales iniciales del sistema.

###  Incluye
- Inserciones (INSERT).
- Datos base.
- Configuraciones iniciales.
- Catálogos.
- Registros necesarios para el funcionamiento del sistema.

###  Objetivo
Restaurar el estado operativo básico de la base de datos.

###  Cuándo usarlo
- Después de ejecutar el respaldo estructural.
- Para recuperar datos esenciales.
- Para pruebas controladas.
